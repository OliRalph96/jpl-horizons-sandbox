# JPL Horizons Sandbox
This work was inspired by a tutorial I found on Youtube:
[Hands On with the Artemis II Trajectory](https://www.youtube.com/watch?v=MJC2sdjua3g) - [Dr. Nora's Guide to the Galaxy](https://www.youtube.com/@NorasGuidetotheGalaxy)

The original data was found here: [Track NASA’s Artemis II Mission in Real Time](https://www.nasa.gov/missions/artemis/artemis-2/track-nasas-artemis-ii-mission-in-real-time/)

And the code used in the tutorial is here: [artemis.py](https://github.com/nora-anne/ngttg/blob/main/artemis.py)

However, the data used in this format no longer exists, so the data was taken from JPL's Horizons System 
which tracks most bodies in the solar system, including satellites such as Artemis II: 
[Horizons Web Application](https://ssd.jpl.nasa.gov/horizons/app.html#/)

Photos of the Horizons settings to reproduce this data can be found in this repo.

There were some challenges with using this data, but I have figured out how to get the equivalent data and put it into the correct format (mostly).
